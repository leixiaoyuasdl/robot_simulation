# RobMach软件

## 所需工具

1.QtCreator

2.Open CasCade-7.4.0-vc14

## 配置环境

1.安装QtCreator（安装中需要勾选MSVC2017）

2.下载Open CasCade进行安装

3.项目.pro文件夹添加相应的路径

![image-1](https://github.com/learnerch/RobMach/blob/master/Images/ReadMe/1.png)
